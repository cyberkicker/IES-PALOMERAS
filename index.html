¿Cómo unirte a nuestro equipo?

Para unirte a nosotros, es fundamental tener un conocimiento básico de GitHub...

A continuación, te proporcionamos un breve resumen de cómo funciona GitHub:

Repositorios:

Espacios donde se almacena el código de un proyecto.

Commits:

Guardan los cambios realizados en el código.

Branches:

Permiten trabajar en diferentes versiones de un proyecto simultáneamente.

Pull Requests:

Proceso para revisar y fusionar cambios propuestos en el código.

¿Cómo empiezo yo por mi cuenta?

Para iniciar, es fundamental tener claros los conceptos básicos de GitHub. Una vez que hayamos accedido a la plataforma, podremos realizar modificaciones en los archivos que hayamos creado, o bien, agregar nuevos archivos utilizando la opción "Upload files". Con el objetivo de desarrollar la página web, es necesario acceder a la sección de "Settings" y luego al apartado de "Pages". Es importante destacar que es esencial tener conocimientos sobre GitHub, ya que el funcionamiento de la página web depende de las ramas (branches) en esta plataforma. Una vez que tengamos la opción de guardar (Save), comenzará a generarse la dirección de dominio de nuestro sitio web. Es importante mencionar que este proceso puede tardar, ya que el servidor suele ser algo lento y puede demorar en actualizar los cambios, a veces más de un minuto.

¿Dónde consigo soporte?

Antes de solicitar soporte, le recomendamos que intente resolver el problema consultando la documentación disponible en nuestros diferentes repositorios o buscando el error específico en Internet.

Si, tras realizar estas búsquedas, no ha logrado encontrar una solución, por favor utilice el apartado de soporte para ponerse en contacto con nosotros. Acceder al soporte
